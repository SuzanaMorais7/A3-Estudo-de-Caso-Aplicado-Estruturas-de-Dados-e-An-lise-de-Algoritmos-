# A3-Estudo-de-Caso-Aplicado-Estruturas-de-Dados-e-An-lise-de-Algoritmos-
Esta atividade tem como objetivo aproximar os estudantes de problemas reais da  computação contemporânea, analisando como empresas e produtos  tecnológicos utilizam análise de algoritmos, estruturas de dados e estratégias de  resolução de problemas complexos para melhorar desempenho, escalabilidade e  viabilidade computacional.
